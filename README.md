# Background information #
This repo contains my script for the analysis of the temeprature range of the White-browed Sparrow weaver in southern Africa. Data used includes research grade observations from iNaturalist and Average global temperature data from WorldClim.org. These data were downloaded into R and used to create interactive maps to make my observation.
A knitted HTML file of my code can be found in this repo that includes my coding process and map outputs.
